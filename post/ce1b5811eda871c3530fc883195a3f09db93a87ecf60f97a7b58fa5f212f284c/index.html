CBS All Access will stop working on older Apple TVs when Paramount Plus launches

CBS All Access is set to transform into Paramount Plus on March 4th, but you won’t be able to watch it if you own a second- or third-generation Apple TV. The company announced that it won’t update its app for those devices when the streaming service relaunches next month, as spotted by MacRumors.

The announcement makes CBS All Access / Paramount Plus the latest service to end support for the set-top box. YouTube announced that it will drop support starting next month, while HBO dropped the aging streaming boxes last year.
