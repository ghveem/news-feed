A modder made a way to mine bitcoin on a Game Boy (very, very slowly)

If your desire to mine bitcoin has been thwarted by a lack of GPU stock and flimsy BIOS locks, YouTuber StackSmashing may be able to help. They’ve made a Game Boy miner, which only needs a Game Boy console (and a slightly modified Link Cable), a Raspberry Pi Pico, and a regular computer (via Tom’s Hardware).

The conceit of the project is that all of the mining happens on a stock Game Boy, which was released a full 20 years before bitcoin was invented. Of course, it does need some help: the Game Boy can’t actually connect to the internet, and it can’t hold the entire blockchain.
